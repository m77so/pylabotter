# PyLabotter
らぼったあ( https://7ka.org/software/labotter/ )のPython劣化移植版

# requirements
- ***Python3.6以上***
- requests==2.18.3
- requests-oauthlib==0.8.0

# 使い方
- labotter.pyを動かす
- localhost:8080にアクセス

# Twitter連携

