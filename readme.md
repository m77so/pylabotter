# PyLabotter
10時押しをするために徹夜をした眠気を紛らわす暇つぶしに作ったらぼったーのPython劣化移植版

# requirements
- *Python3.6以上*
- requests==2.18.3
- requests-oauthlib==0.8.0

# 使い方
- labotter.pyを動かす
- localhost:8080にアクセス

# Twitter連携
config.confとかいうファイルを作って、
```
[twitter]

CS= XXXXXXXXXXXXXXXXXXXXX
AT= 114514-XXXXXXXXXXXXXXXXXXXXXXXx
AS= XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```
って感じで設定してください
